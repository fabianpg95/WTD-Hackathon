Shopping Cart - Add Item

Description: Verify user can add an item to the shopping cart.
Preconditions: User is on the product page.
Steps:
Select a product.
Click on the "Add to Cart" button.
Expected Result: The item is added to the cart and the cart count is updated.
Shopping Cart - Remove Item

Description: Verify user can remove an item from the shopping cart.
Preconditions: User has at least one item in the cart.
Steps:
Go to the shopping cart page.
Click on the "Remove" button next to an item.
Expected Result: The item is removed from the cart and the cart count is updated.
Checkout - Valid Payment

Description: Verify user can complete a purchase with valid payment details.
Preconditions: User has items in the shopping cart.
Steps:
Go to the checkout page.
Enter valid payment details.
Click on the "Place Order" button.
Expected Result: The order is placed successfully and a confirmation message is displayed.
Checkout - Invalid Payment

Description: Verify error message is shown for invalid payment details.
Preconditions: User has items in the shopping cart.
Steps:
Go to the checkout page.
Enter invalid payment details.
Click on the "Place Order" button.
Expected Result: An error message "Invalid payment details" is displayed.

Product Search - Sort by Price

Description: Verify user can sort search results by price.

Steps:
Enter a search query.
Select "Sort by Price" from the sort options.
Expected Result: Search results are sorted by price in ascending order.
